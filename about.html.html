HTML Introduction

What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  1. HTML consists of a series of elements
  2. HTML elements tell the browser how to display the content
  3. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.
